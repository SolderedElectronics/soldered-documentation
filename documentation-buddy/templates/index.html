Welcome

Welcome to Soldered Documentation Buddy! This app is meant to streamline writing documentation for Soldered products. It automates these processes:

  • Creating all the required files for a product group and getting the code to add these pages to the sidebar
  • Creating images of highlighted jumpers and components on the board
  • Generating the 'at a glance' page
  • Generating the 'hardware details' page
  • Partially generating the 'how it works' page
  • Generating the Arduino 'getting started' page
