## AS5600 breakout postition sensor

The **AS5600 breakout postition sensor** is a compact and highly accurate magnetic position sensor module designed for non-contact angle measurement. It enables **360° rotational position sensing**, allowing it to precisely detect the angular position of a rotating shaft or magnet. In addition to angle detection, it features **high-resolution 12-bit output**, providing up to 4096 distinct positions per revolution for smooth and accurate measurements.It features **I2C communication** and is designed for seamless **Qwiic connectivity** — eliminating the need for soldering.

## Key Features

- **Sensing:** 360° non-contact magnetic rotary position sensing
- **Resolution:** 12-bit (up to 4096 positions per revolution)
- **Output modes:** I2C digital, analog voltage, and PWM
- **Communication:** I2C (fixed address: **0x36**, up to 400 kHz)
- **Operating voltage:** 3.3V (onboard regulator for 5V compatibility)
- **Connector:** 2 × **Qwiic ports** (plug-and-play, no soldering needed)
- **Mounting:** **Two mounting holes** for secure attachment
- **Dimensions:** **22 × 22 mm** (0.9 × 0.9 inch)
