## AS5600 breakout postition sensor

The **AS5600 breakout postition sensor** is a compact and highly accurate magnetic position sensor module designed for non-contact angle measurement. It enables **360° rotational position sensing**, allowing it to precisely detect the angular position of a rotating shaft or magnet. In addition to angle detection, it features **high-resolution 12-bit output**, providing up to 4096 distinct positions per revolution for smooth and accurate measurements.It features **I2C communication** and is designed for seamless **Qwiic (formerly easyC) connectivity**—eliminating the need for soldering.

## Key Features
- **Magnetic Rotary Position Sensing**
  - Accurate **360° angle measurement**
  - Non-contact magnetic sensing technology
  - Suitable for rotational position tracking
- **High-Resolution Output**
  - **12-bit resolution**
  - Up to **4096 positions per revolution**
  - Smooth and precise measurements
- **Multiple Output Modes**
  - **I2C digital output**
  - Analog voltage output
  - PWM output for flexible integration
- **I2C-bus Fast Mode Compatible Interface**
  - Fixed address: **0x36**
  - Data rates up to 400 kHz
- **Logic / Operating voltage level:** 3.3V (onboard regulator for 5V compatibility)  
- **Connector:** 2 × **Qwiic (formerly easyC) ports** (plug-and-play, no soldering needed)  
- **Mounting:** **Two mounting holes** for secure attachment  
- **Dimensions:** **22 × 22 mm** (0.9 x 0.9 inch)
