## Inputronic GRID

The **Inputronic GRID** is an interactive module featuring 16 capacitive touch pads and 16 individually addressable WS2812B RGB LEDs arranged in a 4x4 matrix format. It communicates with your microcontroller via **I2C communication**, allowing you to easily read button states and set LED colors or animations. The board is designed for seamless **Qwiic (formerly easyC) connectivity**, enabling simple plug-and-play usage without the need for soldering.

## Key Features

- **Voltage:** 3.3V
- **Communication:** I2C (Default address: `0x30`, easily changeable via software)
- **Inputs:** 16 capacitive touch pads
- **Outputs:** 16 WS2812B-2020 RGB LEDs
- **Microcontroller:** Onboard ATtiny404
- **Connectivity:** Qwiic / easyC connectors for daisy-chaining
