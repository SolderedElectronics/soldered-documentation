# Overview

## NULA Core

Start building your next Arduino-based project with **NULA Core**, a compact yet fully featured development board designed for makers of all skill levels. With **full Arduino IDE compatibility** it brings the classic Arduino Uno experience in a much smaller and more project-friendly size.

Powered by the famous **Atmel ATmega328P**, NULA Core provides 22 I/O pins (14 digital + 8 analog), making it suitable for everything from simple electronics experiments to more advanced projects. Its built-in **USB Type-C connector** ensures easy programming and reliable power delivery.

Designed for modern and fast prototyping, NULA Core includes a **Qwiic connector** for effortless I²C expansion, an onboard 3.3 V regulator for powering sensors and breakouts, and a Li-Ion battery charger with connector for portable applications. It also features a **WS2812B RGB LED**, giving your projects an instant visual upgrade right out of the box.

## Key Features

- **Microcontroller:** ATmega328P (8-bit AVR) @ 16 MHz
- **Operating Voltage:** 5 V logic level
- **GPIO:** 22 pins (14 digital, 8 analog)
- **Communication Interfaces:** UART, SPI, and I²C supported
- **USB-C Connector:** USB Type-C for programming, power and serial communication
- **Battery Support:** Li-Ion battery connector with onboard charging circuit
- **3.3 V Regulator:** Dedicated 3.3 V output for powering sensors and breakouts
- **Compatibility:** 100% compatible with Arduino boards and Arduino IDE
- **Qwiic Connector:** Plug-and-play I²C connector for fast prototyping
- **Onboard Features:** Dedicated User and Reset buttons + onboard WS WS2812B RGB LED
- **Form Factor:** Compact breakout board design (smaller than Arduino Uno)
