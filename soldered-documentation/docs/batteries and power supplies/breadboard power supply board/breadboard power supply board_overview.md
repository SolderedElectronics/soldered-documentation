## Breadboard power supply

The **Breadboard power supply** is a power source for breadboard projects. Connect it via the DC jack or USB-C cable and the board outputs 3.3V, 5V, or 12V directly to the breadboard's power rails. Each output rail is controlled independently with onboard switches. The board also has a power LED, a power switch, and an automatic input selection circuit for when both inputs are connected at the same time.

## Key features

- **Input:** USB-C or DC jack
- **Voltage on DC jack:** 12V DC
- **Voltage on USB-C:** 5V
- **Output current:** 3V3 @ 500mA, 5V @ 1A
- **Dimensions:** 54 x 30mm (2.1 x 1.2 inches)
