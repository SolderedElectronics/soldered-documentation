## Smart LED WS2812B Grid 8x8

The **Smart LED WS2812 Grid 8x8** is a system of 64 daisy-chained **WS2812B RGB LEDs** arranged in a **8x8 grid pattern** that can light up in any color as programmed. Each LED is **individually addressable**, allowing you to to display **animationa, text, symbols, patterns, and dynamic lighting effects** with ease. All Components are placed on a single compact board, making it simple to integrate into your projects and easy to use with microcontrollers such as Arduino, ESP32 or Raspberry Pi.

<InfoBox>
It is possible to connect several modules with just a few wires and control several matrices at once!
</InfoBox>
