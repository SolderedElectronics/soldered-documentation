## Spajanje komponenti

**SHTC3 senzor** koristi **I²C komunikacijski protokol**, što znači da su mu za rad potrebne samo dvije podatkovne linije, **SDA** i **SCL**, te **napajanje** i **masa**.  
Korištenje Qwiic konektora ovo spajanje čini vrlo jednostavnim i preglednim.

| **SHTC3 pin** | **NULA MINI pin** |
|----------------|-------------------|
| Qwiic | Qwiic |

<CenteredImage src="/img/nula-beginner-kit-arduino/DSC00220.JPG" width="1800px" caption="Spajanje SHTC3 senzora preko Qwiic konektora" />

<InfoBox>
Ako tvoj SHTC3 modul nema Qwiic konektor, možeš ga ručno spojiti jumper žicama kao na slici iznad.  
Prije uključivanja napajanja uvijek provjeri jesu li pinovi pravilno spojeni.
</InfoBox>

## Postavljanje biblioteke

U ovom primjeru koristimo **Soldered SHTC3 biblioteku**, koja pojednostavljuje komunikaciju sa senzorom i nudi jednostavne funkcije za čitanje temperature i vlage.

1. Otvori službeni GitHub repozitorij:  
   [**Soldered SHTC3 Library**](https://github.com/SolderedElectronics/Soldered-SHTC3-Temperature-Humidity-Sensor-Arduino-Library)
2. Preuzmi repozitorij kao **ZIP** datoteku.  
3. U Arduino IDE-u idi na **Sketch → Include Library → Add .ZIP Library...**  
4. Pronađi preuzetu ZIP datoteku i potvrdi odabir.  

<CenteredImage src="/img/nula-beginner-kit-arduino/library2.png" alt="Dodavanje biblioteke u Arduino IDE" caption="Dodavanje Soldered SHTC3 biblioteke u Arduino IDE" width="600px"/>

Nakon instalacije, na vrh programa dodaj sljedeći redak:
```cpp
#include "SHTC3-SOLDERED.h"
```

## Inicijalizacija senzora

Prije korištenja SHTC3 senzora moramo ga inicijalizirati pozivom funkcije `begin()` iz biblioteke.  
Biblioteka sama brine o I²C komunikaciji, pa nije potrebno ručno pozivati `Wire.begin()`.  
Ako inicijalizacija ne uspije, u Serial Monitoru će se ispisati poruka o grešci.

```cpp
#include "SHTC3-SOLDERED.h"

SHTC3 shtcSensor;   // Create the sensor object

void setup() {
  Serial.begin(115200);       // Open Serial Monitor for output

  if (!shtcSensor.begin()) {  // Initialize the sensor
    Serial.println("SHTC3 initialization failed!");
  } else {
    Serial.println("SHTC3 sensor ready!");
  }
}
```

---

## Čitanje i prikaz podataka

Funkcija `shtcSensor.sample()` pokreće novo mjerenje, a očitanja zatim dohvaćamo funkcijama `readTempC()` za temperaturu u Celzijevim stupnjevima i `readHumidity()` za relativnu vlažnost u postocima.  
Svake 2 sekunde program uzima novo mjerenje i ispisuje vrijednosti temperature i vlage u Serial Monitor.

```cpp
const unsigned long UPDATE_MS = 2000;
unsigned long lastUpdate = 0;

void loop() {
  unsigned long now = millis();

  if (now - lastUpdate >= UPDATE_MS) {
    lastUpdate = now;

    shtcSensor.sample();                 // Take a new measurement
    float tC  = shtcSensor.readTempC();  // Temperature in °C
    float hum = shtcSensor.readHumidity(); // Humidity in %

    // --- Serial output ---
    Serial.print("Temperature: ");
    Serial.print(tC, 2);
    Serial.print(" °C, Humidity: ");
    Serial.print(hum, 2);
    Serial.println(" %");
  }
}

```
