Cilj ovog primjera je upoznati te sa **serijskom komunikacijom** između pločice **Soldered NULA MINI** i tvog računala. Na taj način pločica može „odgovarati“ natrag, pa možeš otklanjati greške, pratiti vrijednosti senzora ili jednostavno ispisivati tekst.

    - Kako ispisati poruku u Serial Monitor funkcijom `Serial.println()`.

## Što je serijska komunikacija?

Serijska komunikacija je način na koji tvoja NULA MINI pločica i računalo mogu **međusobno komunicirati** pomoću tekstualnih poruka. Zamisli da ti i prijatelj u razredu razmjenjujete papiriće s porukama, ali umjesto da predaš cijelu poruku odjednom, šalješ je **slovo po slovo** ispravnim redoslijedom. To znači serijski prijenos, informacije se šalju jedna za drugom vrlo brzo. Tvoja pločica šalje znakove kao što su 'H','e','l','l','o' tvom računalu, a **Serial Monitor u Arduino IDE-u** ih slaže tako da ti vidiš „Hello“. Ta komunikacija ide preko USB kabela i obje strane se moraju „dogovoriti“ oko brzine (baud rate), kao da se dogovarate da pričate istim jezikom i istim tempom.

---

## Postavljanje serijske komunikacije

Kako bi omogućio serijsku komunikaciju, trebaš je inicijalizirati unutar funkcije **setup()** pozivom
`Serial.begin(baudRate)`. `baudRate` je brzina komunikacije. U ovom primjeru koristi se **115200** bauda. Ova funkcija kaže pločici da počne komunicirati s računalom zadanom brzinom.

```cpp
void setup() {
  /*
  Serial.begin() establishes serial communication between your board and another device, in this
  example, to your computer via a USB cable. In this example we will be communicating with the Serial Monitor, so ensure
  that the baud variable is set to one of the baud rates listed in the Serial Monitor's baud rate dropdown menu. 
  */
  Serial.begin(115200);
}
```
---

## Ispis poruke

Funkciju `Serial.println("text")` koristimo za ispis niza teksta u Serial Monitor, nakon čega slijedi novi red. U ovom primjeru pločica će ispisati „Hello, World!” jedanput, odmah nakon pokretanja. To je zato što se funkcija poziva u setup(). Ako želiš da se poruka ispisuje u petlji, pozovi funkciju unutar loop() funkcije.

<InfoBox>Ako ne želiš prelazak u novi red nakon ispisa teksta, umjesto toga koristi funkciju `Serial.print()`.</InfoBox>

```cpp

void setup() {
  /*
  Serial.begin() establishes serial communication between your board and another device, in this
  example, to your computer via a USB cable. In this example we will be communicating with the Serial Monitor, so ensure
  that the baud variable is set to one of the baud rates listed in the Serial Monitor's baud rate dropdown menu. 
  */
  Serial.begin(115200);

  /*
  Serial.println() Prints data to serial port as human-readable ASCII text followed by a carriage return character ('\r')
  and a newLine character ('\n'). In other words, function prints the text and then skips to the next line.
  */
  Serial.println("Hello, World!");
}

```
