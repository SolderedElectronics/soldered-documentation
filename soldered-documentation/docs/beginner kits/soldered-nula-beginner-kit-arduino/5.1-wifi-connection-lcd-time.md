The goal of this example is to show you how to **connect the NULA RP2040 board to Wi-Fi** and **display the current time** on a **16×2 LCD** using an **NTP (Network Time Protocol)** server.  
This example combines Internet connectivity and display control to create a simple **digital clock** that automatically synchronizes its time online.

<SuccessBox>

**In this documentation you will learn:**
- How to connect the NULA board to a Wi-Fi network using the `WiFi` library.  
- What **NTP (Network Time Protocol)** is and how it provides accurate time.  
- How to use the `NTPClient` library to fetch time from the internet.  
- How to display real-time data on a **16×2 LCD** using the **Soldered LCD Library**.  

</SuccessBox>

## Understanding the concept

This example connects your NULA MINI board to your **Wi-Fi router**, fetches the **current time** from an **NTP server**, and displays it on an LCD.  

Here’s what happens:
1. The NULA MINI board connects to your local Wi-Fi network using your **SSID** and **password**.  
2. It contacts the **NTP server** `pool.ntp.org` to get the current time.  
3. The time is automatically adjusted based on your **timezone offset**.  
4. Every second, the time on the LCD display updates — just like a ticking clock.

## Setting up the code

Start by including all necessary libraries and defining your Wi-Fi credentials:

```cpp
#include "LCD-SOLDERED.h"   // Include the Soldered LCD library
#include <WiFi.h>           // Connect to Wi-Fi
#include <WiFiUdp.h>        // Used by the NTP client
#include <NTPClient.h>      // Get time from the internet

// Replace these with your Wi-Fi network credentials
const char* ssid = "Your SSID";     
const char* password = "Your password";
```

Then, create the objects for the LCD, UDP and NTP client:

```cpp
LCD lcd(16,2);
WiFiUDP ntpUDP;
NTPClient timeClient(ntpUDP, "pool.npt.org", 3600, 60000);
```

Inside `setup()`, we initialize the LCD, display connection progress and connect to the network:

```cpp

void setup(){
    lcd.begin();    //Initialize LCD
    lcd.backlight();    // Turn on backlight
    lcd.setCursor(0,0);
    lcd.print("Connecting WiFi"); 
    WiFi.begin(ssid,password); //Start WiFi connection

    //Wait for successful connection
    while(WiFi.status() != WL_CONNECTED){
        delay(500);
        lcd.setCursor(0,1);
        lcd.print(".");
    }

    lcd.clear();
    lcd.print("WiFi connected");
    delay(1000);
    lcd.clear();¸

    //Start NTP client
    timeClient.begin();
}

```

<InfoBox>If your connection fails, check that your credentials are correct. If the problem persists, try connecting the board to 2.4GHz network. </InfoBox>

## Fetching and displaying time

Once connected, the board continuously updates the time from the NTP server and prints it on the LCD.

```cpp
void loop(){
    timeClient.update();    //Get latest time
    String formattedTime = timeClient.getFormattedTime();   // "HH:MM:SS" format

    lcd.setCursor(0,0);
    lcd.print("Time:");
    lcd.setCursor(0,1);
    lcd.print(formattedTime);
    delay(1000);    //Refresh once per second
}
```

## NTP and time zone

**NTP (Network Time Protocol)** is used to synchronize clocks over the internet. Your NULA MINI bpard connects to a global server pool `pool.ntp.org` to retrieve the current UTC time. To adjust for your local timezone, modify the **offset** in the `NTPClient` constructor:

```cpp
NTPClient timeClient(ntpUDP, "pool.npt.org", timezoneOffset, updateInterval);
```

For example:
-   Central European Time (CET):`timezoneOffset=3600` (1 hour)
-   Eastern Standaard Time (EST):`timezoneOffset=-18000` (-5 hours)
