## Spajanje komponenti

### 1. Spoji LCD

Upotrijebi **Qwiic kabel** da spojiš LCD modul na NULA MINI pločicu.  
Qwiic konektor istim kabelom vodi **napajanje (3.3 V)**, **masu (GND)** i **I²C komunikacijske linije (SDA i SCL)**. Nije potrebno dodatno ožičenje.

## Postavljanje LCD biblioteke

Prije nego počneš koristiti LCD, moraš instalirati **Soldered LCD biblioteku**.  
Ona ti daje jednostavne funkcije za ispis, pozicioniranje i pomicanje teksta.

1. Otvori [https://github.com/SolderedElectronics/Soldered-LCD-Library](https://github.com/SolderedElectronics/Soldered-LCD-Library)  
2. Preuzmi repozitorij kao **ZIP** datoteku  
3. U Arduino IDE-u idi na `Sketch` -> `Include Library` -> `Add .ZIP Library...`  
4. Pronađi i odaberi ZIP datoteku koju si preuzeo  
5. Nakon toga možeš koristiti biblioteku u svojim programima

## Zašto koristiti auto scroll?

**16×2 LCD** može istovremeno prikazati samo 16 znakova u jednom retku.  
Ako ispišeš dužu poruku, ostatak se sprema u memoriju, ali je izvan vidljivog dijela zaslona.  
Funkcija `lcd.scrollDisplayLeft()` pomiče sve vidljive znakove ulijevo za jedan stupac pa tekst izgleda kao da se **pomaknuo**.  
Ponavljanjem tog koraka u petlji dobiješ glatko pomicanje.

Brzina pomicanja ovisi o pauzi između pojedinih koraka.  
Manji `delay()` znači brže pomicanje, a veći `delay()` sporije i čitljivije pomicanje.

## Kod

U nastavku je kompletan primjer koji pomiče poruku preko zaslona.

```cpp

#include "LCD-SOLDERED.h"   // Include the Soldered LCD library

/*
Create an LCD object with 16 columns and 2 rows.
These parameters correspond to your display size.
*/
LCD lcd(16, 2);

/*
Create a message that will scroll across the display.
*/
String message = " Hello from NULA MINI! ";

/*
The speed of scrolling in milliseconds. Higher number = slower scroll.
*/
const int scrollDelay = 300;

void setup() {
  /*
  Initialize the LCD and turn on the backlight.
  */
  lcd.begin();
  lcd.backlight();

  /*
  Print the message once. Only the visible part will show initially.
  */
  lcd.print(message);
}

void loop() {
  /*
  Scroll the text one position to the left, creating a moving effect.
  */
  lcd.scrollDisplayLeft();

  /*
  Wait a short time before the next movement to control scrolling speed.
  */
  delay(scrollDelay);
}
```
