Ovaj primjer započinjemo spajanjem svih komponenti. Samo prati korake u nastavku i sve će proći bez muke.

### 1. Postavi NULA MINI pločicu na breadboard

<InfoBox>Za ovaj korak trebaš poznavati unutarnju strukturu breadboarda i njegove naponske vodove. Ako ti to još nije poznato, pogledaj [**PLACEHOLDER**] stranicu dokumentacije.</InfoBox>

Ako tvoja pločica ima zalemljene muške konektore na pinovima, preporučujemo da je postaviš na breadboard. Tako će ti spajanje ostalih komponenti biti lakše i urednije.

## Dodjeljivanje pinova varijablama

Umjesto da broj pina pišeš izravno u svaki poziv funkcije, možeš ga spremiti u varijablu radi bolje čitljivosti koda. U ovom primjeru varijabla **pinNumber** dobiva vrijednost **4**, što odgovara pinu **IO4** na NULA pločici. Ako kasnije odlučiš spojiti LED na neki drugi pin, trebaš promijeniti samo varijablu na jednom mjestu, a ostatak koda nastavlja raditi.

Možeš koristiti bilo koji pin označen kao IO na pločici. Na slici ispod vidiš sve GPIO pinove koje možeš koristiti na NULA pločici:

<CenteredImage src="/img/nula-beginner-kit-arduino/NULA-MINI-ESP32-C6-Pinout.webp" alt="Svi dostupni GPIO pinovi na NULA MINI" caption="Svi dostupni GPIO pinovi na NULA MINI" width="1800px"/>

Varijable **pinNumber** i **delayMS** deklarirat ćemo prije funkcije **setup()**:

```cpp
/*
This is a variable to which we pass the number of pin that we connected the LED to. 
The NULA board has a pin naming logic as follows: IO4, where 4 is the number that we give to the variable.
If you wish to use a different pin, make sure you are using a IO__ marked pin. 
*/
int pinNumber = 4;

/*
This is a variable that defines the blinking time, in milliseconds.
Feel free to experiment with this value.
*/
int delayMS=1000;

```

## Pauziranje izvođenja programa

Ako bi LED samo vrlo brzo uključivao i isključivao, promjene bi bile toliko brze da ih okom ne bi stigao primijetiti. Da bi treptanje bilo vidljivo, koristi se funkcija `delay()`. Sintaksa je `delay(milliseconds)`, a broj označava koliko dugo program treba pauzirati.

U ovom primjeru varijabla delayMS postavljena je na 1000, što znači da svaka pauza traje jednu sekundu. Kod prvo uključuje LED funkcijom `digitalWrite(pinNumber, HIGH)`, zatim pauzira pomoću `delay(delayMS)`. Nakon toga isključuje LED funkcijom digitalWrite(pinNumber, LOW) i opet pauzira jednu sekundu. Ta se petlja stalno ponavlja i dobivaš jasno vidljivo treptanje.

Funkciju delay() pozivamo nakon svakog poziva digitalWrite():

```cpp
void loop() {

  /*
  digitalWrite() is a function that gives us the ability to change the value that our pin (previously set as OUTPUT pin) 
  writes. As the name of the function tells us, we can only switch between digital values, high and low. Those values
  are represented by different voltage levels, high is 5V while low is 0V. We will start with putting the pin in HIGH
  mode, giving the LED 5V and lighting it up. 
  */
  digitalWrite(pinNumber,HIGH);

  /*
  delay() is a function that starts a pause in the code. Duration of the pause is defined with the ms parameter.
  In this case, we want to wait for a bit after we turned on the LED.
  */
  delay(delayMS);

  //We put the pin in LOW mode, turning the LED off.
  digitalWrite(pinNumber,LOW);

  //Leave the LED turned off for a bit.
  delay(delayMS);
}

```
