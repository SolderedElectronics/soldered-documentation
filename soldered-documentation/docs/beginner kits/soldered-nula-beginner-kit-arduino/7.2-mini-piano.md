Cilj ovog projekta je izraditi **jednostavni mini klavir** koristeći **četiri tipkala** i **pasivni buzzer**.  
Svako tipkalo predstavlja jednu glazbenu notu, a pritiskom na tipkalo svira odgovarajuća frekvencija preko buzzera.  
Projekt spaja pojmove poput **digitalnih ulaza**, **generiranja zvuka** i **upravljanja vremenom**, pa dobivaš osjećaj kako rade jednostavni glazbeni instrumenti.

<SuccessBox>

**U ovoj dokumentaciji naučit ćeš:**
- Kako generirati tonove pomoću funkcija `tone()` i `noTone()`.  
- Kako dodijeliti različite frekvencije više tipkala.  
- Kako spojiti pasivni buzzer i tipkala da dobiješ osnovni klavir.  
- Kako obraditi jednostavnu logiku ulaza s više tipkala.

</SuccessBox>

## Kako radi

**Pasivni buzzer** može proizvoditi tonove različitih frekvencija, što odgovara različitim glazbenim notama.  
Pomoću funkcije `tone()` šaljemo kvadratni signal na buzzer, a frekvencija tog signala određuje visinu tona koji čujemo.  
Svako tipkalo aktivira drugi ton: **C4**, **D4**, **E4** i **F4**.

Kada je tipkalo pritisnuto, svira njegova nota. Kad nijedno tipkalo nije pritisnuto, zvuk se zaustavlja funkcijom `noTone()`.  
U svakom trenutku svira samo jedna nota, prioritet ima prvo detektirano pritisnuto tipkalo.

## Spajanje sklopa

| **Komponenta** | **NULA MINI pin** | **Opis** |
|----------------|-------------------|----------|
| Buzzer | IO18 | Izlazni pin koji generira tonove |
| Tipkalo 1 | IO2 | Svira notu C4 (262 Hz) |
| Tipkalo 2 | IO3 | Svira notu D4 (294 Hz) |
| Tipkalo 3 | IO4 | Svira notu E4 (330 Hz) |
| Tipkalo 4 | IO5 | Svira notu F4 (349 Hz) |

<CenteredImage src="/img/nula-beginner-kit-arduino/DSC00228.JPG" alt="spajanje mini klavira" caption="Raspored spajanja za Mini Piano projekt" width="1800px"/>

Svako tipkalo treba biti spojeno s **pull-down otpornikom** (10 kΩ) kako bi se spriječila lebdeća stanja ulaza.  
Kad se tipkalo pritisne, odgovarajući pin očitava **HIGH** i aktivira svoj ton.

## Kod

U nastavku je prikazan cijeli primjer koda:
```cpp
#define BUZZER_PIN 18

// Button pins
#define BTN1 2
#define BTN2 3
#define BTN3 4
#define BTN4 5

// Frequencies for each button (in Hz)
#define NOTE_C4 262
#define NOTE_D4 294
#define NOTE_E4 330
#define NOTE_F4 349

void setup() {
  // Initialize button pins as inputs with external pull-down ressitors
  pinMode(BTN1, INPUT);
  pinMode(BTN2, INPUT);
  pinMode(BTN3, INPUT);
  pinMode(BTN4, INPUT);

  // Initialize buzzer pin
  pinMode(BUZZER_PIN, OUTPUT);
}

void loop() {
  // Check buttons (active HIGH)
  if (digitalRead(BTN1) == HIGH) {
    tone(BUZZER_PIN, NOTE_C4);
  } 
  else if (digitalRead(BTN2) == HIGH) {
    tone(BUZZER_PIN, NOTE_D4);
  } 
  else if (digitalRead(BTN3) == HIGH) {
    tone(BUZZER_PIN, NOTE_E4);
  } 
  else if (digitalRead(BTN4) == HIGH) {
    tone(BUZZER_PIN, NOTE_F4);
  } 
  else {
    // No button pressed – stop tone
    noTone(BUZZER_PIN);
  }

  delay(50); // Small debounce delay
}

```
