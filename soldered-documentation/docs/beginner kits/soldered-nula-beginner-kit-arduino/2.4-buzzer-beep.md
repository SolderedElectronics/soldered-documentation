Cilj ovog primjera je upoznati te s **generiranjem zvuka** pomoću **pasivnog buzzera** i funkcije `tone()`.  
Pasivni buzzer može proizvoditi zvukove različitih frekvencija, pa možeš puštati tonove, melodije ili zvučne efekte.  
U ovom primjeru tvoja **Soldered NULA MINI** pločica će odsvirati kratku melodiju kada se uključi.

<SuccessBox>
**U ovoj dokumentaciji naučit ćeš:**
- Kako spojiti i upravljati **pasivnim buzzerom**.  
- Kako generirati tonove različitih frekvencija pomoću `tone()`.  
- Kako zaustaviti zvuk pomoću `noTone()`.  
- Kako odsvirati jednostavnu melodiju koristeći polja (arrays).
</SuccessBox>

## Spajanje komponenti

### 1. Spoji buzzer

| **Pin na buzzeru** | **NULA MINI pin** |
|--------------------|-------------------|
| Pozitivni (+)      | IO5               |
| Negativni (–)      | GND               |

<InfoBox>
Pobrini se da koristiš **pasivni buzzer**, a ne aktivni.  
**Pasivni buzzer** može proizvoditi širok raspon frekvencija, dok **aktivni buzzer** može pištati samo na jednom, unaprijed zadanom tonu.
</InfoBox>

## Zvuk i frekvencija

Zvuk nastaje kada nešto vibrira i stvara valove u zraku.  
Ako vrlo brzo prebacujemo digitalni pin između HIGH i LOW stanja (stvori se **kvadratni signal**) određenom frekvencijom, buzzer počinje vibrirati i čujemo ton.

| **Nota** | **Frekvencija (Hz)** |
|----------|----------------------|
| C4       | 262                  |
| D4       | 294                  |
| E4       | 330                  |
| F4       | 349                  |
| G4       | 392                  |
| A4       | 440                  |
| B4       | 494                  |
| C5       | 523                  |

<InfoBox>
Što je frekvencija viša, ton je viši.  
Što je frekvencija niža, ton je dublji.  
</InfoBox>

---

## Kako radi

- Frekvencije pojedinih nota spremljene su u polju `melody[]`.
- Odgovarajuća trajanja nota spremljena su u polju `noteDuration[]`.
- U petlji kod redom svira svaku notu pomoću `tone()`.
- Na kraju melodije buzzer se utišava funkcijom `noTone()`.

---

## Kod

```cpp
/*
This is a variable to which we assign the number of the pin that we connected the buzzer to.
In this example, we’ll use IO5 as the output pin. If you want, you can use any other digital pin that has ADC capabilities.
*/
const int BUZZER_PIN = 5;

/*
We will use two arrays — one for note frequencies (in Hertz) and one for note durations.
These define a short melody that the buzzer will play.
*/
int melody[] = { 262, 294, 330, 349, 392, 440, 494, 523 }; // C4 to C5
int noteDuration[] = { 4, 4, 4, 4, 4, 4, 4, 2 };           // Quarter notes (last one is half note)

void setup() {
  /*
  The tone() function generates a square wave of a given frequency (and duration)
  on a specific pin, which makes the buzzer produce sound.
  */
  pinMode(BUZZER_PIN, OUTPUT);
  Serial.begin(115200);
  Serial.println("Playing melody...");
  
  // Loop through the melody and play each note
  for (int i = 0; i < 8; i++) {
    int duration = 1000 / noteDuration[i];  // Convert duration type to milliseconds
    tone(BUZZER_PIN, melody[i], duration);  // Play the note
    delay(duration * 1.3);                  // Wait a bit before the next note
  }

  // Stop tone when done
  noTone(BUZZER_PIN);
  Serial.println("Melody finished!");
}

void loop() {
  // Nothing happens here — melody plays once in setup()
}
```
