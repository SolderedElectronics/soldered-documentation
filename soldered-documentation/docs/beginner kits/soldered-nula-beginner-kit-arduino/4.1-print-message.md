## Postavljanje LCD biblioteke

Prije nego kreneš programirati ovaj primjer, treba preuzeti **Soldered LCD biblioteku**.  
To je skup već gotovih funkcija koje olakšavaju rad s LCD modulom.

Otvori `https://github.com/SolderedElectronics/Soldered-LCD-Library` i preuzmi repozitorij kao ZIP datoteku.

<CenteredImage src="/img/nula-beginner-kit-arduino/library1.png" alt="Preuzimanje biblioteke" caption="Preuzimanje biblioteke s GitHuba"/>

<InfoBox>Preporučuje se da preuzetu biblioteku premjestiš u poseban folder, a ne da ostane u Downloads mapu.</InfoBox>

Pronađi preuzetu ZIP datoteku biblioteke i potvrdi odabir. Time je biblioteka spremna za korištenje.

## Osnove I2C komunikacije

I2C je način da se elektronički uređaji međusobno **spajaju i komuniciraju koristeći samo dvije žice**:
- **SDA (data line)** nosi podatke.  
- **SCL (clock line)** određuje trenutak kada se podaci šalju i čitaju.

Jedan uređaj je **upravljač** (controller), u ovom slučaju tvoja NULA pločica. Ostali su **periferni uređaji** (peripherals), na primjer LCD zaslon.  
Svaki periferni uređaj ima svoju **adresu**, pa NULA zna kojem uređaju šalje podatke.

Prednost I²C sabirnice je u tome što možeš spojiti **više uređaja na iste dvije žice**, a oni si međusobno neće smetati sve dok imaju različite adrese.

Kada koristiš **Qwiic konektor**, sva potrebna ožičenja (napajanje, masa, SDA i SCL) već su spojena u jednom jednostavnom kabelu, što početnicima bitno olakšava posao.

## Inicijalizacija zaslona

Prije nego što počnemo koristiti LCD biblioteku, moramo je uključiti na vrhu datoteke dodavanjem retka `#include "LCD-SOLDERED.h"`.  
Zatim stvaramo objekt iz klase LCD. U ovom primjeru koristimo konstruktor `lcd(column, row)`.  
Većinu modula, pogotovo Qwiic module, prije upotrebe treba inicijalizirati. To radimo pozivom funkcije `lcd.begin()` unutar funkcije **setup()**.

```cpp
#include "LCD-SOLDERED.h"   // Include the Soldered LCD library

/*
Create an LCD object. 
The parameters (16, 2) mean the LCD has 16 columns and 2 rows.
*/
LCD lcd(16,2);

void setup() {
  /*
  Initialize the LCD so it is ready to use.
  */
  lcd.begin();
}

```

## Prikaz poruke

Sada kada je LCD inicijaliziran, možemo ispisivati tekst pomoću funkcije `lcd.print()`.  
Pomoću `lcd.setCursor(column, row)` određujemo gdje će se poruka pojaviti na zaslonu.  
Gornji lijevi kut zaslona je pozicija (0, 0).

```cpp
void setup(){
  lcd.begin();
  lcd.backlight();

  lcd.clear();           // Clear any previous text
  lcd.setCursor(0, 0);   // Start at the first row, first column
  lcd.print("Hello, Let's");

  lcd.setCursor(0, 1);   // Move to the second row
  lcd.print("get started!");
}

void loop(){
}
```
