Cilj ovog projekta je pokazati kako **proširiti broj digitalnih izlaza** koristeći **74HC595 shift registar**.  
Samo s **tri pina** NULA MINI pločice možeš upravljati **više LED dioda**, što je važan koncept kad radiš s ograničenim brojem I/O pinova.

U ovom primjeru izrađujemo **4-bitni binarni brojač** koji prikazuje vrijednosti od **0 do 15** na četiri LED diode spojene na shift registar.  
Naučit ćeš kako poslati serijske podatke u registar, zaključati (latch) izlaze i u stvarnom vremenu vizualno prikazati binarno brojanje.

<SuccessBox>

**U ovoj dokumentaciji naučit ćeš:**
- Kako **74HC595 shift registar** radi i kako ga spojiti na NULA MINI pločicu  
- Kako koristiti funkciju `shiftOut()` za slanje serijskih podataka  
- Kako upravljati više LED dioda koristeći samo tri izlazna pina  
- Kako vizualizirati binarne brojeve pomoću LED dioda  

</SuccessBox>

## Kako radi

**74HC595** je 8-bitni shift registar sa serijskim ulazom i paralelnim izlazom.  
Omogućuje ti da pošalješ 8 bitova podataka **jedan po jedan** preko jednog podatkovnog pina.  
Kada su svi bitovi učitani, izlaze **zaključavaš (latch)** i u tom trenutku se sve LED diode odjednom ažuriraju.

U ovom projektu NULA MINI šalje 4-bitne vrijednosti (0–15) koje predstavljaju **binarni brojač**.  
Svaka LED dioda odgovara jednom bitu tog broja:

| Decimal | Binary | LED uzorak (Q3 Q2 Q1 Q0) |
|---------|--------|--------------------------|
| 0 | 0000 | Sve isključene |
| 1 | 0001 | Uključena samo desna LED |
| 7 | 0111 | Uključene tri desne LED |
| 15 | 1111 | Sve uključene |

## Spajanje i pinovi

| **74HC595 pin** | **Funkcija** | **NULA MINI pin** | **Opis** |
|-----------------|-------------|-------------------|----------|
| DS (pin 14) | Serijski podatkovni ulaz | IO2 | Podaci koje šalje NULA MINI |
| SH_CP (pin 11) | Clock | IO4 | Pomiče svaki bit u registar |
| ST_CP (pin 12) | Latch | IO3 | Ažurira izlaze prema LED diodama |
| Q0–Q3 (pinovi 15,1,2,3) | Izlazi | 4 LED diode (preko 220 Ω otpornika) | Prikazuju binarnu vrijednost |
| MR (pin 10) | Master reset | 5 V | Drži registar aktivnim |
| OE (pin 13) | Output enable | GND | Omogućuje prikaz izlaza |
| VCC (pin 16) | Napajanje | 5 V | Napajanje sklopa |
| GND (pin 8) | Masa | GND | Zajednička masa |

## Cijeli kod

U nastavku je kompletan primjer koda za ovaj projekt:
```cpp
int latchPin = 3;  // ST_CP pin (Latch)
int clockPin = 4; // SH_CP pin (Clock)
int dataPin = 2;  // DS pin (Data)

int counter = 0;   // 4-bit counter (0–15)

void setup() {
  pinMode(latchPin, OUTPUT);
  pinMode(clockPin, OUTPUT);
  pinMode(dataPin, OUTPUT);
}

void loop() {
  // Limit counter to 4 bits (0–15)
  byte value = counter & 0x0F;  

  digitalWrite(latchPin, LOW);    // Prepare shift register for data
  shiftOut(dataPin, clockPin, MSBFIRST, value);  // Send data bits
  digitalWrite(latchPin, HIGH);   // Latch outputs to LEDs

  counter++;  // Increment counter
  if (counter > 15){
    counter = 0;  // Wrap around to 0 after 15
  } 

  delay(500);  // Wait half a second
}
```
