The goal of this example is to show you how to use an **ultrasonic distance sensor** to control a **buzzer**.  
The closer an object gets to the sensor, the faster the buzzer beeps — just like a parking sensor in a car.

<SuccessBox>

**In this documentation you will learn:**
- How to connect the **HC-SR04 ultrasonic sensor** and **buzzer** to the NULA board.  
- How to measure distance using the **Soldered Ultrasonic Sensor library**.  
- How to make a buzzer beep at different speeds depending on the measured distance.  
- How to use simple conditions and `tone()` function to generate sound.

</SuccessBox>

**Hardware required:**
- 1× Soldered NULA MINI board  
- 1× Soldered Ultrasonic Distance Sensor HC-SR04  
- 1× Passive buzzer
- 1× Breadboard  
- Jumper wires  
- USB-C cable

### 1. Connect the ultrasonic sensor

| **Sensor Pin** | **NULA MINI Pin** |
|----------------|-------------------|
| VCC | 3.3V |
| GND | GND |
| TRIG | IO04 |

## How it works

The ultrasonic sensor measures distance by sending a sound wave and waiting for it to bounce back.  
The **closer** an object is, the **shorter** the echo time.  

In this example, we use that distance to control **how fast the buzzer beeps**:  
- When the object is **far**, the beeps are **slow**.  
- When the object is **close**, the beeps are **fast and continuous**.

This mimics a **proximity alert system**.

---

## Setting up the code

Include the **Soldered Ultrasonic Sensor** library and define the pins for both the ultrasonic sensor and the buzzer.

```cpp
#include "Ultrasonic-distance-sensor-easyC-SOLDERED.h"   // Include the Soldered ultrasonic sensor library

// --- Pin definitions ---
#define TRIGPIN   4     // Trigger pin for the sensor
#define ECHOPIN   3     // Echo pin for the sensor
#define BUZZERPIN 2     // Buzzer output pin

// --- Create ultrasonic sensor object ---
Ultrasonic_Sensor hc(TRIGPIN, ECHOPIN);
```

Then initialize everything inside **setup()**:
```cpp
void setup(){
    Serial.begin(115200);
    hc.begin();
    pinMode(BUZZERPIN, OUTPUT);

}
```

---

## Measuring distance and controlling the buzzer

We measure the distance usng the library's `getDistance()` function and adjust the buzzer's beeping interval accordingly.

```cpp
void loop(){
    float distance=hc.getDistance();
    Serial.print("Distance: ");
    Serial.print(distance);
    Serial.println("cm");

    if (distance > 50 || distance <= 0) {
    // If object is too far or not detected, turn buzzer off
    noTone(BUZZERPIN);
  } 
  else if (distance > 20) {
    // Medium range: slow beeps
    tone(BUZZERPIN, 1000, 100);  // Play 1kHz tone for 100 ms
    delay(400);                  // Pause between beeps
  } 
  else if (distance > 10) {
    // Close: faster beeps
    tone(BUZZERPIN, 1500, 100);
    delay(200);
  } 
  else {
    // Very close: continuous tone
    tone(BUZZERPIN, 2000);       // Constant 2kHz tone
    delay(50);
  }

  noTone(BUZZERPIN);  // Stop tone before next measurement
}
```

<InfoBox> `tone(pin, frequency, duration)` plays a sound of the given frequency (Hz) for a certain time (ms). `noTone(pin)` stops the sound on that pin. </InfoBox>
