Cilj ovog primjera je pokazati kako **mjeriti udaljenost** pomoću **ultrazvučnog senzora HC-SR04** i NULA MINI pločice.  
Senzor koristi **ultrazvučne valove** da bi odredio koliko je neki objekt udaljen, tako da mjeri vrijeme potrebno da impuls zvuka dođe do objekta i vrati se natrag.  
Izračunata udaljenost prikazuje se u **centimetrima** u **Serial Monitoru**.

<SuccessBox>
**U ovoj dokumentaciji naučit ćeš:**
- Kako radi **HC-SR04 ultrazvučni senzor**.  
- Kako spojiti senzor i očitavati podatke pomoću NULA MINI pločice.  
- Kako izračunati udaljenost koristeći **brzinu zvuka**.  
- Kako prikazati rezultat u Serial Monitoru.
</SuccessBox>

## Spajanje komponenti

### 1. Spoji ultrazvučni senzor

HC-SR04 ima **četiri pina**: **VCC**, **TRIG**, **ECHO** i **GND**.  
Senzor spajamo na NULA MINI pločicu kako je prikazano u tablici.

| **HC-SR04 pin** | **NULA MINI pin** |
|------------------|-------------------|
| VCC | 3.3V |
| TRIG | IO4 |
| ECHO | IO3 |
| GND | GND |

<InfoBox>
Senzor HC-SR04 je u pravilu predviđen za rad na **5 V**, ali može raditi i na **3.3 V**, samo s nešto manjim dosegom.  
Pobrini se da su **TRIG** i **ECHO** izravno spojeni na GPIO pinove NULA MINI pločice, tako da nije potreban level shifter.
</InfoBox>

## Kako radi ultrazvučni senzor

**HC-SR04** mjeri udaljenost koristeći zvučne valove koji se šire kroz zrak.  
Kada program pokrene mjerenje, pin **TRIG** šalje kratak ultrazvučni impuls, niz zvučnih valova frekvencije oko **40 kHz**, što je iznad granice ljudskog sluha. Ti valovi putuju naprijed dok ne naiđu na neki objekt. Kada dotaknu površinu, odbijaju se natrag prema senzoru, a pin **ECHO** bilježi njihov povratak.

Senzor zatim mjeri vrijeme između slanja i primanja zvučnog impulsa.  
Budući da je **brzina zvuka** u zraku otprilike **343 metra u sekundi**, odnosno oko `0.0343` cm/µs, možemo izračunati udaljenost koju je zvuk prešao:

`Distance = (Duration * 0.0343) / 2`

<InfoBox>Zvučni val mora putovati do objekta i natrag do senzora, zato izmjereno vrijeme dijelimo s dva kako bismo dobili udaljenost u jednom smjeru.</InfoBox>

## Kod

U nastavku je prikazan kompletan primjer koda koji mjeri udaljenost i ispisuje je u Serial Monitor pri brzini od 115200 baud.

```cpp
#include "Ultrasonic-distance-sensor-easyC-SOLDERED.h"   // Include Soldered Ultrasonic Sensor library

// --- Pin definitions ---
#define TRIG_PIN  4     // Trigger pin for the ultrasonic sensor
#define ECHO_PIN  3     // Echo pin for the ultrasonic sensor

// --- Create ultrasonic sensor object ---
Ultrasonic_Sensor hc(TRIG_PIN, ECHO_PIN);

void setup() {
  Serial.begin(115200);   // Start Serial Monitor communication
  hc.begin();             // Initialize the ultrasonic sensor

  Serial.println("Ultrasonic Measuring Distance Example started!");
  Serial.println("Move an object in front of the sensor to see the distance change.");
}

void loop() {
  // --- Measure distance in centimeters ---
  float distance = hc.getDistance();

  // --- Print distance to Serial Monitor ---
  Serial.print("Distance: ");
  Serial.print(distance, 2);
  Serial.println(" cm");

  delay(500);  // Short delay between readings
}
```
