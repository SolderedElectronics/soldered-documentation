Cilj ovog projekta je napraviti **pametni RGB LED kontroler** koji reagira na ambijentalno svjetlo pomoću **fotootpornika (LDR)**.  
Kako se razina svjetla mijenja, **boja LED diode se glatko mijenja** kroz spektar: od **crvene u mraku**, preko **zelene i plave**, pa sve do **bijele na jakom svjetlu**.  
Projekt spaja **analogni ulaz** i **PWM izlaz** te pokazuje kako senzore i rasvjetu možeš povezati u dinamične vizualne efekte.

<SuccessBox>

**U ovoj dokumentaciji naučit ćeš:**
- Kako koristiti **fotootpornik (LDR)** za mjerenje ambijentalne razine svjetla.  
- Kako koristiti `map()` za skaliranje analognih očitanja na svjetlinu LED diode.  
- Kako ostvariti glatke **prijelaze boja** pomoću PWM izlaza.  
- Kako izgraditi RGB LED kontroler punog spektra koristeći samo jedan analogni senzor.

</SuccessBox>

## Kako radi

**Fotootpornik** mijenja svoj otpor ovisno o količini svjetla:  
- jako svjetlo → manji otpor → veće analogno očitanje  
- slabo svjetlo → veći otpor → manje analogno očitanje  

NULA MINI čita taj analogni signal (0–4095) pomoću `analogRead()` i zatim ga **preslikava na intenzitet RGB LED kanala** pomoću funkcije `map()`.  
Ovisno o razini svjetla, LED dioda se glatko mijenja između crvene, zelene, plave i na kraju bijele, stvarajući dinamičan “light mood” efekt.

## Spajanje i ožičenje

| **Komponenta** | **NULA MINI pin** | **Opis** |
|----------------|------------------|----------|
| LDR (fotootpornik) | IO5 | Analogni ulaz (spožen u djelitelj napona s otpornikom od 10 kΩ) |
| Crvena nožica RGB LED | IO2 | PWM izlaz za crveni kanal |
| Zelena nožica RGB LED | IO3 | PWM izlaz za zeleni kanal |
| Plava nožica RGB LED | IO4 | PWM izlaz za plavi kanal |
| Zajednička katoda (GND LED) | GND | Zajednički spoj na masu |
| LDR s otpornikom 10 kΩ | 3.3V -> LDR -> pin > otpornik -> GND | Čini analogni djelitelj napona |

## Cijeli kod

U nastavku je potpuni primjer koda za ovaj projekt:
```cpp
#define LDR_PIN   5   // Analog input pin for photoresistor
#define RED_PIN   2   // PWM pin for red LED channel
#define GREEN_PIN 3   // PWM pin for green LED channel
#define BLUE_PIN  4   // PWM pin for blue LED channel

int ldrValue = 0;     // Photoresistor reading (0–4095)
int r = 0, g = 0, b = 0;

void setup() {
  Serial.begin(115200);
  pinMode(LDR_PIN, INPUT);
  pinMode(RED_PIN, OUTPUT);
  pinMode(GREEN_PIN, OUTPUT);
  pinMode(BLUE_PIN, OUTPUT);

  Serial.println("RGB LED Controller with full-spectrum color mapping started");
}

void loop() {
  // --- Read light level from photoresistor ---
  ldrValue = analogRead(LDR_PIN);
  Serial.print("LDR value: ");
  Serial.println(ldrValue);

  // --- Map LDR value (0–4095) to RGB channels ---
  if (ldrValue <= 1365) {
    // Dark → transition from Red (255,0,0) to Green (0,255,0)
    r = map(ldrValue, 0, 1365, 255, 0);
    g = map(ldrValue, 0, 1365, 0, 255);
    b = 0;
  }
  else if (ldrValue <= 2730) {
    // Medium → transition from Green (0,255,0) to Blue (0,0,255)
    r = 0;
    g = map(ldrValue, 1366, 2730, 255, 0);
    b = map(ldrValue, 1366, 2730, 0, 255);
  }
  else {
    // Bright → transition from Blue (0,0,255) to White (255,255,255)
    r = map(ldrValue, 2731, 4095, 0, 255);
    g = map(ldrValue, 2731, 4095, 0, 255);
    b = 255;
  }

  // --- Write PWM values to RGB pins ---
  analogWrite(RED_PIN, r);
  analogWrite(GREEN_PIN, g);
  analogWrite(BLUE_PIN, b);

  // --- Debug output ---
  Serial.print("RGB: ");
  Serial.print(r); Serial.print(", ");
  Serial.print(g); Serial.print(", ");
  Serial.println(b);

  delay(100);
}
```
