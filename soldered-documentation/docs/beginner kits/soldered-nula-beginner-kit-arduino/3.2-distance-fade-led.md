Cilj ovog primjera je pokazati kako pomoću **ultrazvučnog senzora udaljenosti** možeš upravljati **svjetlinom LED diode**.  
Što je objekt bliže senzoru, LED svijetli jače.  
Ovaj projekt spaja **mjerenje udaljenosti** i **PWM upravljanje LED diodom** pa možeš vidjeti podatke sa senzora kroz promjenu intenziteta svjetla.

<SuccessBox>

**U ovoj dokumentaciji naučit ćeš:**
- Kako spojiti **HC-SR04 ultrazvučni senzor** na NULA MINI pločicu.  
- Kako mjeriti udaljenost koristeći **Soldered Ultrasonic Sensor** biblioteku.  
- Kako koristiti funkciju `map()` za pretvaranje udaljenosti u svjetlinu.  
- Kako upravljati svjetlinom LED diode pomoću PWM signala (`analogWrite()`).  
- Kako istovremeno ispisivati udaljenost i vrijednost svjetline u Serial Monitor.

</SuccessBox>

**Potreban hardver:**
- 1× Soldered NULA MINI pločica  
- 1× HC-SR04 ultrazvučni senzor  
- 1× LED dioda (bilo koje boje)  
- 1× otpornik od 330 Ω  
- 1× breadboard  
- jumper žice  
- USB-C kabel

## Spajanje komponenti

### 1. Spoji LED diodu

Krećemo tako da LED spojimo na isti način kao u primjeru [**1.1 LED Blinking**](/documentation/soldered-nula-beginner-kit-arduino/led-blinking).  

- Spoji **anodu** (dulja nožica) LED diode preko **otpornika od 330 Ω** na **IO02** na NULA MINI pločici.  
- Spoji **katodu** (kraća nožica) izravno na **GND**.

<CenteredImage src="/img/under_construction.png" alt="privremena slika" caption="Spojena LED dioda na IO2 i GND preko otpornika" width="600px"/>

### 2. Spoji ultrazvučni senzor

| **HC-SR04 pin** | **NULA MINI pin** |
|------------------|------------------|
| VCC | 3.3V |
| GND | GND |
| TRIG | IO04 |
| ECHO | IO03 |

<InfoBox>
Pin **TRIG** šalje kratki ultrazvučni impuls i tako pokreće mjerenje.  
Pin **ECHO** prima reflektirani signal i mjeri koliko je vremena prošlo dok se zvučni val nije vratio.  
Za što točnije očitanje usmjeri senzor prema ravnoj površini, na primjer zidu.
</InfoBox>

## Kako radi ultrazvučni senzor?

**HC-SR04** mjeri udaljenost tako da pošalje kratki niz ultrazvučnih valova i zatim mjeri koliko je vremena potrebno da se vrate nakon odbijanja od objekta.  
Budući da je **brzina zvuka** u zraku oko **343 m/s**, senzor može izračunati udaljenost tako da vrijeme putovanja pomnoži s tom brzinom i podijeli s dva (put naprijed i natrag).

Biblioteka Soldered Ultrasonic Sensor taj postupak pojednostavljuje.  
Umjesto da ručno mjeriš trajanje impulsa, samo pozoveš `getDistance()` i odmah dobiješ udaljenost u centimetrima.  
Ako želiš vidjeti i „sirovo“ vrijeme odjeka, možeš pozvati `getDuration()`.

## PWM i upravljanje svjetlinom

NULA MINI ne generira pravi analogni napon, već koristi **PWM (Pulse Width Modulation)**.  
Kod PWM-a se pin vrlo brzo uključuje i isključuje, a podešavanjem vremena koliko je pin uključen u odnosu na isključen (duty cycle) mijenjamo koliko LED izgleda svijetlo.

| **Duty cycle** 	| **Vizualni efekt** 	|
|---	|---	|
| 0% 	| LED je potpuno ugašena 	|
| 50% 	| LED svijetli srednjim intenzitetom 	|
| 100% 	| LED svijetli punim intenzitetom 	|

<CenteredImage src="/img/nula-beginner-kit-arduino/pwm-explained.png" alt="PWM objašnjen" caption="Prikaz PWM signala i svjetline LED diode" />

Funkcija `analogWrite(pin, value)` postavlja duty cycle.  
Što je vrijednost veća, LED dioda svijetli jače.

## Kod

U nastavku je prikazan kompletan primjer koji čita udaljenost sa senzora i prema njoj podešava svjetlinu LED diode.

```cpp
#include "Ultrasonic-distance-sensor-easyC-SOLDERED.h"  // Include Soldered Ultrasonic Sensor library

// --- Pin definitions ---
#define TRIG_PIN  4     // Trigger pin for the sensor
#define ECHO_PIN  3     // Echo pin for the sensor
#define LED_PIN   2     // LED connected to IO2 (PWM-capable pin)

// --- Create ultrasonic sensor object ---
Ultrasonic_Sensor hc(TRIG_PIN, ECHO_PIN);

void setup() {
  Serial.begin(115200);  // Start Serial communication
  pinMode(LED_PIN, OUTPUT);

  hc.begin();  // Initialize ultrasonic sensor

  Serial.println("Distance Fade LED Example started!");
}

void loop() {
  // --- Measure distance ---
  float distance = hc.getDistance();  // Distance in centimeters
  long duration  = hc.getDuration();  // Echo round-trip time in microseconds (for info)

  // --- Print distance to Serial Monitor ---
  Serial.print("Distance: ");
  Serial.print(distance, 1);
  Serial.print(" cm");

  // --- Limit distance range to useful values ---
  if (distance < 2) distance = 2;
  if (distance > 50) distance = 50;

  // --- Map distance to LED brightness ---
  // Closer object = higher brightness
  int brightness = map(distance, 2, 50, 4095, 0);
  analogWrite(LED_PIN, brightness);

  // --- Print brightness level ---
  Serial.print(" -> Brightness: ");
  Serial.println(brightness);

  delay(100);
}
```
