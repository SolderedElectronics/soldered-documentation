Cilj ovog primjera je naučiti te kako koristiti ulazni način rada pinova na pločici **Soldered NULA MINI**. Ulazi omogućuju pločici da reagira na vanjske signale, kao što su tipkala, prekidači ili senzori. U ovom primjeru koristit ćemo **tipkalo** kao jednostavan ulazni uređaj za brojanje pritisaka. Upoznat ćemo i pojam **debouncea** i kasnije ga primijeniti u kodu.

<SuccessBox>
**U ovoj dokumentaciji naučit ćeš:**
- Kako spojiti tipkalo na NULA pločicu pomoću **pull-down otpornika**.  
- Kako čitati stanje tipkala funkcijom `digitalRead()`.  
- Kako povećavati brojač pri svakom pritisku i ispisivati rezultat u Serial Monitor.  
- Zašto jedan pritisak tipkala može uzrokovati više povećanja brojača.  
</SuccessBox>

## Spajanje komponenti

Ovaj primjer započinjemo spajanjem svih komponenti. Samo prati korake u nastavku i sve će proći bez puno muke.

### 1. Postavi NULA MINI pločicu na breadboard

<InfoBox>Za ovaj korak trebaš poznavati unutarnju strukturu breadboarda i njegove naponske vodove. Ako ti to još nije poznato, pogledaj [**PLACEHOLDER**] stranicu dokumentacije.</InfoBox>

Ako tvoja pločica ima zalemljene muške konektore na pinovima, preporučujemo da je postaviš na breadboard. Tako će spajanje ostalih komponenti biti lakše i urednije.

Ako nema, nema problema. Pločicu i ostale komponente možeš povezati i običnim žicama. Bit će malo neurednije, ali će sasvim dobro raditi.

<InfoBox>  
### Što je pull-down otpornik?  

Kada tipkalo spojiš izravno na pin **NULA MINI** pločice, pin ne zna uvijek treba li čitati stanje **HIGH** ili **LOW** dok tipkalo nije pritisnuto. U tom „lebdećem“ stanju pin može hvatati električni šum i nasumično prelaziti između HIGH i LOW, što dovodi do nepredvidivog ponašanja.  

**Pull-down otpornik** rješava taj problem tako da lagano „povlači“ vrijednost pina dolje na **LOW (0 V)** kada tipkalo nije pritisnuto. Možeš ga zamisliti kao oprugu koja uvijek vraća pin na masu kada ništa drugo ne djeluje na njega.  

- Kada tipkalo **nije pritisnuto**, otpornik spaja pin na **GND**, pa NULA pločica očitava **LOW**.  
- Kada je tipkalo **pritisnuto**, tipkalo spaja pin izravno na **3.3 V** i pločica očitava **HIGH**.  

Vrijednost otpornika (najčešće **10 kΩ**) bira se tako da bude dovoljno jak da drži pin u stanju LOW kada je sklop u mirovanju, ali dovoljno slab da ne ometa struju kada je tipkalo pritisnuto.  

Bez pull-down otpornika ulaz s tipkala može se ponašati nepredvidivo i ponekad pokazivati HIGH čak i kada tipkalo ne pritišćeš.  
</InfoBox>

## Čitanje stanja tipkala

Za čitanje stanja tipkala koristimo funkciju `digitalRead()`. Važno je da je pin podešen kao **INPUT**, inače ova funkcija neće raditi ispravno.

Funkcija `digitalRead(pin)` čita vrijednost sa zadanog digitalnog pina i kao rezultat vraća **HIGH** ili **LOW**. Funkciju pozivamo unutar funkcije **loop()**:

```cpp

void loop() {
  
  //digitalRead() is a function that reads the value from a specified digital pin, either HIGH or LOW.
  bool reading = digitalRead(BUTTON_PIN);
}
```

## Kako radi

Ovaj primjer pokazuje kako **čitati digitalni ulaz** i reagirati na pritisak tipkala.  
Svaki put kada tipkalo prijeđe u stanje HIGH, povećavamo varijablu brojača za 1 i ispisujemo vrijednost u **Serial Monitor**.

Budući da ovaj primjer **ne koristi debounce**, primijetit ćeš da jedan pritisak može dati više povećanja brojača.  
To je namjerno i na tom problemu ćemo raditi u [**2.2 Button Debounce**](/documentation/soldered-nula-beginner-kit.arduino/button-debounce/).

## Kod

U nastavku je prikazan puni kod za ovaj primjer. Kontinuirano čita ulaz s tipkala i ispisuje koliko je puta tipkalo pritisnuto.

```cpp
/*
This is a variable to which we pass the number of the pin that we connected the BUTTON to.
The NULA board has a pin naming logic as follows: IO4, where 19 is the number that we give to the variable.
If you wish to use a different pin, make sure you are using a IO__ marked pin.
*/
const int BUTTON_PIN = 4;

/*
This variable holds the value of our counter.
Each time the button is pressed, the counter value will increase by one.
*/
int counter = 0;

void setup() {

  /*
  pinMode() is a function that configures the specified pin to behave either as an input or an output.
  As our pin needs to detect if the button has been pressed, we will put the pin in INPUT mode.
  */
  pinMode(BUTTON_PIN, INPUT);

  /*
  Serial.begin() initializes serial communication between the NULA board and the computer.
  We use it here so we can print out the current counter value on the Serial Monitor.
  */
  Serial.begin(115200);

  /*
  Print out the initial message so we know that the program started successfully.
  */
  Serial.println("Button Counter Example started!");
  Serial.println("Press the button to increase the counter...");
}

void loop() {

  /*
  digitalRead() is a function that reads the value from a specified digital pin, either HIGH or LOW.
  When the button is pressed, it connects the pin to 3.3V, and the board reads HIGH.
  When it’s released, the pin is connected to GND through a pull-down resistor and reads LOW.
  */
  bool reading = digitalRead(BUTTON_PIN);

  /*
  If the button is pressed, increase the counter by one and print it to the Serial Monitor.
  Since this version does not include debouncing, multiple counts may appear for a single press.
  */
  if (reading == HIGH) {
    counter++;
    Serial.print("Counter: ");
    Serial.println(counter);

    /*
    Wait for the button to be released before allowing another count.
    This prevents the counter from increasing too quickly while the button is still held down.
    */
    while (digitalRead(BUTTON_PIN) == HIGH) {
      // Wait until the button is released
    }
  }
}

```
