## Introduction

The **Macro Pad Solder Kit** is a fun and educational electronics kit that teaches you the fundamentals of soldering and electronics while allowing you to build a functional custom device. For those unfamiliar with a macropad, it is a compact peripheral device equipped with programmable keys (macros), used to trigger shortcuts or automated commands. Using the onboard **RP2040 microcontroller**, each key can be programmed to perform specific **actions** or execute **sequences of commands**. In addition, the rotary encoder can also be programmed, allowing it to control functions such as volume adjustment, scrolling or other custom inputs. All this makes the macro pad a useful tool for speeding up repetitive task and improving work efficiency.

## Contents of the kit

<CenteredImage src="/img/macro_pad/macro_pad_solder_kit_assembly.JPG" alt="Assembled macro pad image" caption="Macro Pad Solder Kit" />

- **PCB:** Macro Pad circuit board with pre-soldered microcontroller, USB-C connector, and other SMD components
- **Microcontroller:** RP2040 dual-core ARM Cortex-M0+ (133 MHz, 264 KB SRAM), fully reprogrammable via USB-C
- **5 x MX switches**
- **5 x Keycaps (for the MX switches)**
- **Rotary encoder (cap included)**
- **1 x LED**
- **Resistors:**
  - 8 x 10 kΩ
  - 2 x 470 Ω
  - 2 x 1 kΩ
  - 1 x 2.2 kΩ
- **Capacitor 0.1 µF**
- **1 x pushbutton**
- **Male Header pins**
