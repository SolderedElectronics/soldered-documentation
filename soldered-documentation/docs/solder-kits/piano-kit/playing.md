## Playing notes

After plugging in the finished board to USB-C for power toggle the **POWER** switch to make sure it's on:

<CenteredImage src="/img/piano-solder-kit/on_off.jpg" alt="On/Off button" caption="Power switch"/>

Press the capacitive touch buttons to play corresponding piano notes:

<CenteredImage src="/img/piano-solder-kit/notes.jpg" alt="Notes" caption="Capacitive touch keys corresponing to Piano notes"/>

You can adjust the total output volume using the volume potentiometer: 

<CenteredImage src="/img/piano-solder-kit/volume.jpg" alt="Volume" caption="Volume potentiometer"/>

## Octave selection

Using the octave selection, you can change the octave you're playing in, 4 different octaves are supported:

<CenteredImage src="/img/piano-solder-kit/octave.jpg" alt="Octave" caption="Octave selection switch"/>

<InfoBox>Try changing the octaves on the fly as you're playing!</InfoBox>

## Arpeggio mode

The built-in arpeggio mode can be enabled by using the **ARPEGGIO** switch, and the speed between changing notes is determined by the **ARPEGGIO SCALE** potentiometer:

<CenteredImage src="/img/piano-solder-kit/arp_1.jpg" alt="Arpeggio settings" caption="Arpeggio settings"/>

After this mode is enabled, holding down multiple keys will change between the pressed automatically in an **ascending** manner.
