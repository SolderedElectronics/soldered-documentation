## Introduction

**Lo-Fi Noise Machine Kit** is a fun, educational electronics project that teaches you the fundamentals of soldering, and at the end you get a really cool sound device! This kit lets you build your own **analog noise machine**, creating **Lo-Fi** tones with no **digital sound processing** or **software involved**. It features a buzzer, an 8Ω speaker, and a 3.5mm audio jack, giving you the flexibility to choose between three different sound output options depending on how you want to use it.

## Contents of the kit

- **NE556N dual precision timer**
- **2 x potentiometers** for controlling pitch and width
- **3.5 mm audio jack**
- **USB Type C connector**
- **0.5W 8Ohm speaker**
- **Buzzer**
- **Voltage regulator**
- **Capacitors:**

  - 1 x 150R 1W

  - 1 x 6.2k
