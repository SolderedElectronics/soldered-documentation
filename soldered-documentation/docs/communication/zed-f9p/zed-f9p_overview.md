## GPS-GNSS BREAKOUT ZED-F9P

The **GPS-GNSS BREAKOUT ZED-F9P** features the **u-blox F9 receiver platform**, delivering **multi-band GNSS** capabilities for high-volume industrial applications. The module integrates **u-blox multi-band RTK** and **PPP-RTK** technologies, enabling **centimeter-level positioning accuracy**.

It also supports **moving base functionality**, allowing both the base and rover to move while continuously computing their relative position. Communication is available via UART or I2C, selectable using the JP1 jumper, ensuring easy integration with microcontrollers such as **Arduino**.

This makes the module a strong choice for outdoor navigation and positioning projects where a high level of precision is required.

## Key Features

- **Position Accuracy:** centimeter-level
- **Multiple GNSS Support:** Works with GPS, GLONASS, Galileo and BeiDou.
- **Multi-band RTK and PPP-RTK support**
- **Moving base functionality**
- **Communication:** UART/SPI/I2C
- **Operating Voltage:** 3.3V
